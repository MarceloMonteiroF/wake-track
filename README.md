**# Wake Track**



**## 📌 Objetivo**



**O Wake Track é um sistema de gerenciamento de academias desenvolvido para facilitar o controle de alunos, professores e treinos. O sistema tem como foco centralizar informações, disponibilizar planilhas de treinamento e melhorar a comunicação entre academia, professores e alunos.**



**---**



**## 👥 Atores do Sistema**



**### Academia**



**\* Cadastrar alunos**

**\* Cadastrar professores**

**\* Gerenciar informações da academia**



**### Professor**



**\* Criar e editar treinos**

**\* Acompanhar o desempenho dos alunos**

**\* Registrar observações e recomendações**



**### Aluno**



**\* Visualizar treinos**

**\* Registrar observações sobre a execução dos exercícios**

**\* Acompanhar orientações dos professores**



**---**



**## 🚀 Principais Funcionalidades**



**### Gestão de Usuários**



**\* Cadastro de alunos**

**\* Cadastro de professores**



**### Gestão de Treinos**



**\* Criação de treinos**

**\* Edição de treinos**

**\* Visualização de treinos**



**### Acompanhamento**



**\* Registro de observações pelo professor**

**\* Registro de observações pelo aluno**

**\* Histórico de treinos**
